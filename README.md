# CRUD App using Node.js, Express & MongoDB

## 📌 Project Description
This is a simple CRUD (Create, Read, Update, Delete) application built using:
- HTML
- CSS
- Bootstrap
- JavaScript
- Node.js
- Express.js
- MongoDB

## 🚀 Features
- Add new data
- View all records
- Update existing data
- Delete records

## 🛠 Technologies Used
- Frontend: HTML, CSS, Bootstrap
- Backend: Node.js, Express
- Database: MongoDB

## 📂 Project Structure
- models/
- public/
- server.js
