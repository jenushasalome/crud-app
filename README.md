# CRUD App using Node.js, Express & MongoDB

## 📌 Project Description
This is a simple CRUD (Create, Read, Update, Delete) application built using Node.js, Express and MongoDB.  
Users can create, read, update and delete student records.

---

## 🚀 Features
- Add new student
- View all students
- Update student details
- Delete student records

---

## 🛠 Technologies Used
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## ⚙️ How to Run the Project Locally

1. Clone the repository:
   git clone https://github.com/jenushasalome/crud-app.git

2. Navigate to project folder:
   cd crud-app

3. Install dependencies:
   npm install

4. Create a .env file and add:
   MONGO_URI=your_mongodb_connection_string
   PORT=5000

5. Start the server:
   node server.js

6. Open browser and go to:
   http://localhost:5000
